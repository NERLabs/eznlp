## 关键发现

1. **专家词典效果最佳**: 手工标注的专家词典达到 97.941%，显著优于其他方法
2. **软词典表现中等**: SoftLexicon 达到 95.88%，优于基线但低于专家词典
3. **词典质量关键**: 手工词典 > 自动词典 > 软词典，质量直接影响性能
