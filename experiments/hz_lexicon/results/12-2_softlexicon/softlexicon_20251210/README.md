## 实验配置

### 模型架构
- **预训练模型**: hfl/chinese-macbert-base
- **编码器**: BiLSTM (hidden_dim=256, num_layers=1)
- **解码器**: CRF
- **特征**: SoftLexicon 软词典特征

### 训练参数
- **训练轮数**: 30 epochs
- **批次大小**: 16
- **学习率**: 
  - BERT: 2e-5
  - 其他参数: 2e-3
- **权重衰减**: 1e-4
- **梯度裁剪**: 5.0
- **Dropout**: 0.5

### 词典配置
- **软词典维度**: 从训练集构建
- **特征类型**: BMES 软匹配特征

## 关键发现

1. **性能表现**
   - 测试集 F1 达到 95.88%
   - 验证集最佳 F1 达到 96.96%
   - 相比旧版 BERT+SoftLexicon (95.07%) 提升 0.81%

2. **与其他方法对比**
   - 手工专家词典: 97.941% (**最佳**)
   - 自动专家词典: 97.050%
   - SoftLexicon: 95.88% (**本实验**)
   - 历史 SoftLexicon: 95.07%

3. **训练特点**
   - 训练过程稳定，收敛良好
   - 第22轮验证集达到最佳
   - 后期存在轻微过拟合现象
