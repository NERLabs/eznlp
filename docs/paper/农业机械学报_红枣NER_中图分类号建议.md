# 中图分类号建议

生成日期：2026-05-23

## 1. 建议候选

优先建议：

```text
TP391.1
```

备选：

```text
TP391
```

## 2. 理由

本文研究对象是红枣栽培文本中的命名实体识别，核心方法属于中文文本信息处理、自然语言处理、知识抽取和农业知识图谱基础技术。公开中图分类号查询页面显示：

1. `TP39` 为“计算机的应用”。
2. `TP391` 为“信息处理（信息加工）”。
3. `TP391.1` 为“文字信息处理”。

因此，若投稿系统允许细分到四级类目，`TP391.1` 与本文“中文农业技术文本命名实体识别”的主题更贴合；若投稿系统或模板只要求较粗粒度类号，可填 `TP391`。

## 3. 不建议作为唯一主分类的类号

`S126` 可覆盖农业信息化相关主题，但本文方法和实验核心是文本实体识别算法，不是农业信息系统建设、农业信息服务平台或农业生产信息化装备，因此不建议将 `S126` 作为唯一主分类号。若编辑部希望突出农业信息技术方向，可考虑：

```text
TP391.1; S126
```

但官方核对表要求中图分类号不超过 2 个，最终仍需作者按《中国图书馆分类法》正式表确认。

```text
TP391.1
```

终稿前由作者或所在单位图书馆/编辑部确认。
